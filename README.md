# 27.7.1
добавление логгера в серверную часть чата


в прикрепренных файлах две части чата:
Серверная
Клиентская

Логгер выполняет запись служебных сообщений сервера в отдельный файл log.txt в папку LOG по расположению проекта, а также чтение строк из файла лога и вывод их в консоль, таким образом сервер не только обрабатывает команды от клиента, но и визулизирует их обработку в консоли
